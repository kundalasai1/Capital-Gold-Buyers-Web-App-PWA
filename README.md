This is the **Professional Development Package** application for **Capital Gold Buyers**. It is a full-stack, responsive web application and Progressive Web App (PWA) built using Next.js (App Router), TypeScript, Tailwind CSS, and Prisma ORM.

## Features Checklist

- **Live Gold Rate Management:** Public rates widget with automatic API synchronization hooks and manual admin rate edits with change history auditing.
- **Gold Value Calculator:** Instant payout calculator (8% branch assay deduction fee) with opt-in lead capture.
- **Appointment Booking System:** Dynamic time-slot selector based on local store branch hours and active schedules.
- **Lead Management Dashboard:** Admin panel to search, filter, assign, and add timestamped follow-up logs for leads.
- **Blog CMS:** CRUD dashboard for education posts with auto-generated SEO slugs and tags.
- **Branch Management:** Store location manager for addresses, hours, map links, and visibility controls.
- **Email Notifications:** Automated branded HTML transaction receipt emails dispatched via Nodemailer.
- **Direct Agent Call:** A floating widget with browser calling simulation, call timers, agent priority queues, and callback requests.
- **Progressive Web App (PWA):** Installs standalone on devices, precaches pages, provides offline fallback rate cards.
- **Security & SEO:** Custom cookie session JWT auth, role restrictions, LocalBusiness schemas, robots.txt, and dynamic sitemaps.

## Directory Architecture

- `/prisma`: Holds SQLite schema model files and seed data.
- `/public`: Static assets, manifest.json, and PWA Service Worker (sw.js).
- `/src/app`: App Router pages and API routing directories.
  - `/api`: REST API routes (Auth, Rates, Leads, Booking, Calls, CMS, Analytics).
  - `/admin`: Role-restricted staff portal views.
  - `/blog`, `/calculator`, `/book`, `/branches`, `/contact`, `/offline`: Public layouts.
- `/src/components`: Reusable UI modules (GoldRateWidget, CallAgentWidget, PWARegistration, Navbar, Footer).
- `/src/lib`: Server utility modules (Prisma client, JWT auth helpers, nodemailer mailer, calling adapters).

## Setup & Running Locally

### 1. Prerequisites
- Node.js (v18.0.0 or later)
- npm (v9.0.0 or later)

### 2. Install Dependencies
```bash
npm install
```

### 3. Initialize the Database
This project uses SQLite for zero-configuration local development.
```bash
# Run database migrations and sync schema
$env:DATABASE_URL="file:./dev.db"; npx prisma migrate dev --name init

# Generate Prisma Client classes
$env:DATABASE_URL="file:./dev.db"; npx prisma generate

# Seed default store data, staff users, and spot rates
$env:DATABASE_URL="file:./dev.db"; node prisma/seed.js
```

### 4. Running the Dev Server
To run locally, you can pass the SQLite URL inline:
```bash
$env:DATABASE_URL="file:./dev.db"; npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables Config

Create a `.env` file at the root of the project with these exact keys. Leave values blank as they will be configured manually:

| Key | Description | Where to Obtain |
|---|---|---|
| `DATABASE_URL` | Prisma DB connection URL (e.g. `file:./dev.db` for SQLite) | Local directory path, or PostgreSQL link (Neon/Supabase) for production |
| `NEXTAUTH_SECRET` | Secret key used to encrypt custom JWT cookies | Random 32-character string generated by `openssl rand -hex 32` |
| `EMAIL_PROVIDER_API_KEY` | Mail client API key for SMTP dispatch | SMTP provider API key (e.g., Resend, Sendgrid, custom SMTP pass) |
| `GOOGLE_ANALYTICS_ID` | Google Analytics GA4 Measurement ID | Google Analytics Admin dashboard (format: `G-XXXXXXXXXX`) |
| `GOOGLE_MAPS_API_KEY` | Google Maps API key for address geolocations | Google Cloud Platform Console maps credentials |
| `TWILIO_ACCOUNT_SID` | Twilio Account identifier for browser calls | Twilio Console Dashboard |
| `TWILIO_AUTH_TOKEN` | Twilio Auth token key | Twilio Console Dashboard |
| `TWILIO_VOICE_NUMBER` | Twilio phone line for agent calling queues | Twilio Console bought phone number |
| `NEXT_PUBLIC_VAPID_KEY` | Public VAPID key for web push alerts | Web Push VAPID key generator |
